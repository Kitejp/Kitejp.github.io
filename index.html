Game1

ゲーム概要
タイトル Hero's Adventure
ジャンル 2Dアクションゲーム
制作期間 2020年12月～2021年6月,2021年9月20日～
制作人数 1人
制作環境 Unity 2019.4.16.f1, Visual Studio 2019 Community
使用言語 C#

制作理由

私が初めてゲームをプレイしたとき、プレイヤーを自分であるかのように自由に動かせるということにとても感動し、将来はゲームプログラマーになり、プレイヤーの動きを作りたいと思うようになりました。
アクションゲームは、他のジャンルのゲームと比べてプレイヤーの動きが非常に重要なゲームであると考えており、アクションゲームを制作することで、プレイヤーに関する知識・技術が効率良く身に着けられると思ったため、このゲームを制作しました。

力を入れて取り組んだこと・工夫した点

制作理由にもあるように、プレイヤーの動きに関する技術を身に着けたかったので、プレイヤーの動きに力を入れて取り組みました。
操作に必要なキーは全部で７つだけですが、ジャンプのキーを入力する長さによって、飛ぶ高さが変わったり、壁キックを出来るようにすることでよりアクションゲームらしさを出せるように努力しました。

改善点

・敵のHPが0の時になり、敵のオブジェクトが消える前に攻撃をすると、2回倒した判定になってしまうので、修正する。
・ステージ数を増やしたいと思います。
・ジャンプの高さの変化を活かせるようなレベルデザインをしたいと思います。

ダウンロードはこちら！

https://github.com/Kitejp/Portfolio

Game2

ゲーム概要
タイトル 虫食い算in富山
ジャンル 計算パズルゲーム
制作期間 2021年1月～
制作人数 2人
制作環境 Unity 2019.4.16.f1, Visual Studio 2019 Community
使用言語 C#

担当箇所

UI設計、ランキングシステムの実装

制作理由

こちらのゲームは、「つくるUOZUゲームジャム-Winter-」というゲームジャムで友人と一緒に制作したゲームになります。
高齢者の方々向けのゲームを制作してほしいというお題でしたので、高齢者の認知症予防のために、過度な「頭の体操」が出来るものを作りました。他のゲームにはない、筆算の虫食い算にすることでオリジナリティを出しました。
また、こちらのゲームは、富山県高齢福祉課様と契約を結び、本格的なゲームにするために、現在も制作を続けております。

力を入れて取り組んでいること

高齢者の方へ向けたゲームですので、「文字を見やすくする」、「制限時間などは文字以外のもので表現する」、「VFXを活用する」、「BGM、SEを加える」などで、高齢者の方でも分かりやすくなるように制作をしております。
今後は、キーボードの操作に慣れていない方のために、仮想キーボードを実装する予定です。

Game3

ゲーム概要
タイトル どれだけ多く敵を倒せるか競うゲーム（仮）
ジャンル 3Dアクションゲーム
制作期間 2021年6月～2021年8月
制作人数 1人
制作環境 Unity 2020.3.2.f1, Visual Studio 2019 Community
使用言語 C#

制作理由

こちらのゲームは、学校の課題で制作した3Dのアクションゲームになります。
このゲームを制作しようと思ったきっかけですが、当時の私は、少しストレスが溜まっており、何かストレスを発散出来るものを探していたのですが、なかなか見つかりませんでした。そこで、「自分で作ってみればいいのではないか」と思い、制作してみることにしました。

工夫した点

3Dのゲームは、2Dのゲームと違い、プレイヤーの操作に、視点移動も加わってくるため、スムーズにプレイするためには、慣れが必要かと思います。そこで、敵にロックオン出来るようにすることで、敵を倒しやすくしました。

Game4

ゲーム概要
タイトル ブロックパズルゲーム（仮）
ジャンル パズルゲーム
制作期間 2021年8月31日～
制作人数 1人
制作環境 Visual Studio 2019 Community
使用言語 C++
使用ライブラリ DXライブラリ

制作理由

C++とDXライブラリを用いたゲームプログラミングの技術を向上させるために、テトリスのようなブロックパズルゲームを制作しました。

工夫した点

クラスやファイルの分割をするなどオブジェクト指向を活用し、また、コメントで自分の意図を書くことで、他の方にも見やすく、理解しやすいコードを書くことを意識しました。

改善点

今のままでは、ユーザー様に何回も遊んでいただく工夫が足りないと感じたため、ランキングを実装し、他の方と競えるようにすることで、より長く遊んでもらえるようにしたいです。
